## "12개 MCP" 정의

강의 12개 = **Ch1 3 + Ch2 4 + Ch3 3 + Ch4 2**

| Ch | MCP |
|---|---|
| 1. 데이터 | ① Google Sheets ② GA4 ③ Firecrawl |
| 2. 콘텐츠·영상·디자인 | ④ Figma ⑤ YouTube Data ⑥ Higgsfield ⑦ 영상제작 |
| 3. 배포·광고 | ⑧ Buffer ⑨ Meta Ads ⑩ Google Ads |
| 4. 협업·운영 | ⑪ Notion ⑫ Discord |

> ⑦ 영상제작은 도구 3개(Hyperframes+HeyGen+ElevenLabs)를 1개로 묶은 것 → **실제 설치 MCP 서버는 13개**. 폴더는 `00-MCP란` 인트로 포함 12개.

## 스킬 3가지 모드 (호출 시 가장 먼저 물음)

| 모드 | 동작 | 언제 |
|---|---|---|
| **A. 실제 설치** | 미설치만 설치, 완료된 건 자동 스킵 | 처음 환경 셋업 |
| **B. 교육 시연** ⭐ | **이미 다 깔려 있어도** 12개 전 과정을 1개씩 설명·검증 (비파괴) | **강의 녹화·복습** |
| **C. 점검만** | 12개 현재 상태만 표로 출력 | 빠른 헬스체크 |

> 이미 모든 MCP 가 설치된 PC 에서 **설치 과정 전체를 보여줘야 할 때**는 **B 교육 시연 모드**.
> 기존 `.env`·인증을 덮어쓰지 않고(비파괴), 각 단계마다 ① 무엇·왜 → ② 발급 절차 → ③ 설치 명령 → ④ 읽기전용 헬스체크로 "지금 작동 중" 증명. 호출: "MCP 설치 전 과정 보여줘".
